# vue-firebase-test
Simple project used to learn how to use vuejs with firebase.

## Prerequisites

* It was developed with [**node**](https://nodejs.org/) version `12.16.1`.

### Getting Started
Inside the _frontend_ folder run:

```
npm install
npm run serve
```
